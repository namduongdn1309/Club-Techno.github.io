## Headings
```md
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

## Text Formatting
```md
**Bold**
*Italic*
***Bold Italic***
~~Strikethrough~~
`Inline code`
```

## Blockquotes
```md
> Blockquote
>> Nested blockquote
```

## Lists

### Unordered
```md
- Item
- Item
  - Subitem
```

### Ordered
```md
1. First
2. Second
3. Third
```

## Task Lists
```md
- [x] Done
- [ ] Todo
```

## Links
```md
[GitHub](https://github.com)
```

## Images
```md
![Alt text](https://via.placeholder.com/150)
```

## Code Blocks

### Fenced with Language
```md
```js
console.log("Hello World");
```
```

## Tables
```md
| Name | Type | Status |
|------|------|--------|
| API  | REST | Stable |
| UI   | Web  | Beta   |
```

## Horizontal Rule
```md
---
```

## Collapsible Sections
```md
<details>
<summary>Click to expand</summary>

Hidden content here

</details>
```

## Emojis
```md
:rocket: :sparkles: :bug: :warning:
```

## Mentions & Issues
```md
@username
#123
```

## Keyboard Keys
```md
<kbd>Ctrl</kbd> + <kbd>C</kbd>
```

## Footnotes
```md
Here is a note[^1]

[^1]: Footnote text
```

## Badges
```md
![License](https://img.shields.io/badge/license-MIT-green)
```

<b>Bold</b>

```

```html

```

```md

```
